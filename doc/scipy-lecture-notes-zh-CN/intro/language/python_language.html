1.2. Python 语言¶

Authors: Chris Burns, Christophe Combelles, Emmanuelle Gouillart, Gaël Varoquaux

Python – 科学计算

此处我们仅提供了有关Scipy和Numpy密切相关的部分知识.想要更加深入地学习，请参考官方文档 https://docs.python.org/tutorial. 同时也有一些额外的书籍可以参考 http://www.diveintopython.net/.

小技巧

Python 是一门 编程语言, 和 C, Fortran, BASIC, PHP, 等一样. 下面列出了一些Python的特性:

  • 它是一门 解释语言 (同 编译语言 相对) . 与 C 和 Fortran 等语言相比 Python代码执行之前不需要编译。除此之外，Python代码也可以 交互式 执行。
  • 它是一门 open-source 协议下的免费软件: Python可以免费地使用，分发， 甚至是用于商业目的。
  • 跨平台: Python支持所有主流操作系统，如 Windows, Linux/Unix, MacOS X, most likely your mobile phone OS 等。
  • 简洁的，具有很高可读性的代码格式
  • 拥有很多的可用于各种用途的扩展包，从网站建设到科学计算。
  • 可以很方便地接入其它编程语言 C 和 C++ 。
  • 其他：Python是一门面向对象的动态语言。

详情参见 https://www.python.org/about/。
