csv
	逗号分隔值文件格式，其文件以纯文本形式存储表格数据
	
	纯文本意味着该文件是一个字符序列，不含必须像二进制数字那样被解读的数据
	
	csv 文件由任意数目记录组成，记录间以某种换行符分割
	
	使用工具：记事本
	
	规则
		1. 开头不留空，以行为单位
		2. 可含或不含列名，含列名则居文件第一行
		3. 一行数据不跨行，无空行
		4. 以半角逗号作为分隔符

数据驱动测试
	相同的测试脚本使用不同的测试数据来执行，
	测试数据和测试行为完全分离，
	这样的测试脚本设计模式称为数据驱动
	
	实施数据驱动测试步骤：
		1. 编写测试脚本，能够从外部读取数据
		2. 测试数据导入到外部介质中
		3. 运行脚本，观察外部介质中的测试数据被调用
		4. 验证所有的测试结果是否符合预期结果
	
	与 excel 文件不同，csv 文件中：
		1. 内容全是字符串，没有其他数据类型
		2. 没有颜色和样式，不能指定单元格宽高，不能合并单元格
		3. 没有多个工作表
		4. 不能嵌入图像图标
